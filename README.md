## Pre requisites

1. Node.js >= v6.10.3 (Download at https://nodejs.org)
2. npm >= v5.0.0 () 

## Run
1. git clone https://github.com/aas919/palindrome.git
2. cd palindrome
3. npm install
4. ng serve
5. Visit http://localhost:4200
